六、发帖
http://m.gkong.com/API/BBS_New_Topic.ashx
输入
{
  UID: 登录验证票,
  Title: 帖子标题,
  Body: 帖子内容,
  BoardId: 版块ID
}


七、回帖
http://m.gkong.com//API/BBS_Repay_Topic.ashx
输入
{
  UID: 登录验证票,
  AnnounceId: ID,
  Body: 帖子内容,
  IsIncRepay: bool是否为引用
}



八、最新回帖列表，最新发帖列表
ht://api.gkong.com/serviceapi.svc/GetBBSListNew
输入JSON格式  Type, Sort, Page
{
  Type: "tech"                    // "tech" 技术区帖子   "notech" 非技术区帖子
  Sort: "date",                   // "date" 按最后回复时间排序  "id" 按发帖时间排序
  Page: "1"                       // 页码
}
九、工控搜索接口
http://s.gkong.com/Service/wcfSearch.svc/Search

输入JSON格式  
{
  name: "搜索论坛",                //固定值
  query: "变频器",                 //关键词
  page: 1                          //页码
}
输出JSON格式
{
  PageInfo: Object,               //页码信息
  SearchList: Object              //搜索结果列表
}
